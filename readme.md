#### Installation

No such Installation is required. Although to process code following commands are used:- 

###### To Compile SASS

`npm run compile:sass`

###### To Comile and Watch SASS

`npm run start`

###### To Concat CSS

`npm run concat:css`

###### To Prefix CSS

`npm run prefix:css`

###### To Compress CSS

`npm run compress:css`

###### To Build Css

i.e To Concat, Prefix & Compress CSS

`npm run build:css`
